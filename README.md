# Deteksi Cacat Citra Radiografi Pengelasan

Aplikasi ini merupakan implementasi model YOLOv8 Segmentation untuk mendeteksi cacat pada citra radiografi hasil pengelasan.

## Fitur

- Upload citra radiografi
- Deteksi otomatis menggunakan YOLOv8
- Menampilkan status inspeksi
- Menampilkan jenis cacat yang terdeteksi
